# Tilemap Isométrico

Mapa isométrico (losango) feito em OpenGL.

## Como executar

Basta rodar o executável já incluído na pasta:

```
.\iso-tilemap.exe
```

Para recompilar (opcional):

```
mingw32-make
```

## Controles

Navegação entre os tiles (8 direções):

```
        W
     Q     E
   A           D
     Z     C
        S
```

As **setas** do teclado também movem o cursor.

- `1`..`7` → troca o tipo do tile atual
- `ESPAÇO` → cicla para o próximo tipo
- `ESC` → sair
